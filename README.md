# Resumer

Resumer is an AI-powered application that helps you customize your resume and generate cover letters for job applications. Built with modern web technologies, Resumer aims to simplify the job application process by providing a user-friendly interface and powerful features.

## Features

- **AI-Powered Resume Customization**: Tailor your resume to specific job applications using AI.
- **Cover Letter Generation**: Automatically generate personalized cover letters.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **User-Friendly Interface**: Easy navigation and intuitive design.

## Technologies Used

- **TypeScript**
- **JavaScript**
- **React**
- **npm**

## Project Structure

- `src/components/Header.tsx`: Contains the header component with navigation links.
- `src/components/Footer.tsx`: Contains the footer component with branding and copyright.
- `index.html`: The main HTML file that includes the root element and links to the main script.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/resumer.git
    ```
2. Navigate to the project directory:
    ```sh
    cd resumer
    ```
3. Install the dependencies:
    ```sh
    npm install
    ```

### Running the Application

To start the development server, run:
```sh
npm start
```
The application will be available at `http://localhost:3000`.

### Building for Production

To create a production build, run:
```sh
npm run build
```
The build output will be in the `build` directory.
